# Customer-Shopping-Analytics-Dashboard-SQL-Power-BI

End-to-End Data Analytics Project that analyzes customer shopping behavior using MySQL for data analysis and Power BI for interactive dashboard visualization. The project transforms raw customer shopping data into meaningful business insights through SQL queries, KPIs, and dynamic dashboards.

Project Overview

This project focuses on analyzing customer purchasing behavior to help businesses understand sales performance, customer demographics, shopping preferences, and purchasing trends.

The analysis was performed using MySQL, while the interactive dashboard was built in Power BI to provide business stakeholders with actionable insights.
